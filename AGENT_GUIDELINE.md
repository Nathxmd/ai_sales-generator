# Agent Guideline: AI Sales Page Generator
> **Option B** — Laravel 11 + Inertia.js + React + PostgreSQL + Gemini API  
> Deadline: 30 April 2026 | Stack: Full-stack monolith with modern SPA feel

---

## 1. Project Overview

Build a web application that transforms raw product/service information into a complete, styled sales page using Google Gemini AI. The output must render as a real landing page — not raw text.

**Core deliverables (non-negotiable):**
1. User Authentication (register, login, logout)
2. Product Input Form (6 fields)
3. AI-Powered Sales Page Generation via Gemini
4. Saved Pages with view, re-generate, delete
5. Live Preview as styled landing page
6. Export as `.html` file

## 9. GeminiService (Core AI Logic)

**File:** `app/Services/GeminiService.php`

> ⚠️ ALL Gemini API logic must live in this file. Never call the API from controllers.

```php
<?php

namespace App\Services;

use Illuminate\Support\Facades\Http;
use Illuminate\Support\Facades\Log;
use Exception;

class GeminiService
{
    private string $apiKey;
    private string $apiUrl;

    public function __construct()
    {
        $this->apiKey = config('services.gemini.key');
        $this->apiUrl = config('services.gemini.url');
    }

    public function generateSalesPage(array $productData): array
    {
        $prompt = $this->buildPrompt($productData);

        $response = Http::timeout(30)->post("{$this->apiUrl}?key={$this->apiKey}", [
            'contents' => [
                [
                    'parts' => [
                        ['text' => $prompt]
                    ]
                ]
            ],
            'generationConfig' => [
                'temperature' => 0.7,
                'maxOutputTokens' => 2048,
            ]
        ]);

        if (!$response->successful()) {
            Log::error('Gemini API error', ['status' => $response->status(), 'body' => $response->body()]);
            throw new Exception('AI generation failed. Please try again.');
        }

        $text = $response->json('candidates.0.content.parts.0.text');

        if (!$text) {
            throw new Exception('Empty response from AI. Please try again.');
        }

        return $this->parseJsonResponse($text);
    }

    private function buildPrompt(array $data): string
    {
        $featuresText = is_array($data['features'])
            ? implode(', ', $data['features'])
            : $data['features'];

        return <<<PROMPT
You are an expert marketing copywriter. Generate a complete, persuasive sales page for the following product.

Return ONLY a valid JSON object with exactly these keys (no markdown, no explanation, no code block):
{
  "headline": "A compelling, benefit-driven headline (max 12 words)",
  "sub_headline": "Supporting headline that adds context (max 20 words)",
  "description": "2-3 paragraph product description, persuasive and clear",
  "benefits": ["benefit 1", "benefit 2", "benefit 3", "benefit 4"],
  "features": [
    {"title": "Feature Name", "detail": "One sentence explanation"},
    {"title": "Feature Name", "detail": "One sentence explanation"},
    {"title": "Feature Name", "detail": "One sentence explanation"}
  ],
  "social_proof": "A realistic placeholder testimonial with name and role",
  "pricing_text": "Pricing summary text (mention the price and value)",
  "cta_text": "Call-to-action button text (max 6 words)"
}

Product name: {$data['product_name']}
Description: {$data['description']}
Key features: {$featuresText}
Target audience: {$data['target_audience']}
Price: {$data['price']}
Unique selling points: {$data['usp']}

IMPORTANT: Return ONLY the JSON object. No markdown. No ```json. No explanation before or after.
PROMPT;
    }

    private function parseJsonResponse(string $text): array
    {
        // Strip any markdown code fences if Gemini adds them
        $clean = preg_replace('/```json|```/i', '', $text);
        $clean = trim($clean);

        $decoded = json_decode($clean, true);

        if (json_last_error() !== JSON_ERROR_NONE) {
            Log::error('JSON parse failed', ['raw' => $text]);
            throw new Exception('AI returned invalid format. Please regenerate.');
        }

        // Validate required keys exist
        $required = ['headline', 'sub_headline', 'description', 'benefits', 'features', 'social_proof', 'pricing_text', 'cta_text'];
        foreach ($required as $key) {
            if (!isset($decoded[$key])) {
                throw new Exception("AI response missing required field: {$key}");
            }
        }

        return $decoded;
    }
}
```

## 10. SalesPageController

**File:** `app/Http/Controllers/SalesPageController.php`

```php
<?php

namespace App\Http\Controllers;

use App\Http\Requests\GeneratePageRequest;
use App\Models\SalesPage;
use App\Services\GeminiService;
use Illuminate\Http\Request;
use Inertia\Inertia;

class SalesPageController extends Controller
{
    public function __construct(private GeminiService $gemini) {}

    // Show the input form
    public function create()
    {
        return Inertia::render('SalesPages/Create');
    }

    // Store input + trigger AI generation
    public function store(GeneratePageRequest $request)
    {
        $page = SalesPage::create([
            ...$request->validated(),
            'user_id' => auth()->id(),
            'features' => array_filter(array_map('trim', explode(',', $request->features))),
            'status' => 'pending',
        ]);

        try {
            $content = $this->gemini->generateSalesPage($page->toArray());
            $page->update(['generated_content' => $content, 'status' => 'generated']);
        } catch (\Exception $e) {
            $page->update(['status' => 'failed']);
            return back()->withErrors(['ai' => $e->getMessage()]);
        }

        return redirect()->route('pages.show', $page)->with('success', 'Sales page generated!');
    }

    // Show generated page (live preview)
    public function show(SalesPage $page)
    {
        $this->authorize('view', $page); // ensure user owns this page
        return Inertia::render('SalesPages/Show', ['page' => $page]);
    }

    // History list
    public function history(Request $request)
    {
        $pages = auth()->user()->salesPages()
            ->when($request->search, fn($q) => $q->where('product_name', 'ilike', "%{$request->search}%"))
            ->latest()
            ->paginate(10)
            ->withQueryString();

        return Inertia::render('SalesPages/History', [
            'pages' => $pages,
            'filters' => $request->only('search'),
        ]);
    }

    // Delete
    public function destroy(SalesPage $page)
    {
        $this->authorize('delete', $page);
        $page->delete();
        return redirect()->route('pages.history')->with('success', 'Page deleted.');
    }

    // Re-generate AI content for existing input data
    public function regenerate(SalesPage $page)
    {
        $this->authorize('update', $page);

        try {
            $content = $this->gemini->generateSalesPage($page->toArray());
            $page->update(['generated_content' => $content, 'status' => 'generated']);
        } catch (\Exception $e) {
            return back()->withErrors(['ai' => $e->getMessage()]);
        }

        return redirect()->route('pages.show', $page)->with('success', 'Page regenerated!');
    }
}

class ExportController extends Controller
{
    public function html(SalesPage $page)
    {
        $this->authorize('view', $page);

        if ($page->status !== 'generated') {
            return back()->withErrors(['export' => 'This page has no generated content yet.']);

        $filename = str($page->product_name)->slug() . '-sales-page.html';

        return response($html)
            ->header('Content-Type', 'text/html')
            ->header('Content-Disposition', "attachment; filename=\"{$filename}\"");
    }

## 15. Inertia — Data Flow Rules

> Agent must follow this pattern. Never create separate REST API endpoints for frontend data.

```
User Action (React)
  → Inertia form submit / router.post / router.delete
    → Laravel Controller (handles logic, calls GeminiService)
      → Returns Inertia::render('ComponentName', ['data' => $data])
        → React receives as props automatically
```

**Do not:**
- Create `api/` routes for frontend
- Use `axios` or `fetch` to call your own backend
- Use `useState` to store server data — use Inertia props instead

**Do:**
- Use `useForm` from `@inertiajs/react` for all forms
- Use `router.post/delete/patch` for non-form actions
- Pass all page data from controller via `Inertia::render()`

## 16. Error Handling Checklist

Every AI generation flow must handle these cases:

| Scenario | Handling |
|---|---|
| Gemini API timeout (>30s) | `Http::timeout(30)` + catch Exception → show user error |
| Gemini returns non-JSON | `parseJsonResponse()` strips fences + validates |
| Missing JSON keys | Explicit key check in `parseJsonResponse()` |
| API quota exceeded | HTTP 429 → log + return friendly message |
| User not owner of page | Policy check → 403 |
| Page has no content (status: failed) | Show fallback UI, offer regenerate button |
